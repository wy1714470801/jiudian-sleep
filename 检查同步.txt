# 检查步骤

1. 查看最后一次提交的内容
git log -1 --stat

2. 查看远程仓库状态
git remote -v

3. 确认本地和远程是否同步
git fetch origin
git status

4. 如果不同步，强制推送
git push -f origin main

5. 访问 GitHub 查看文件内容
https://github.com/wy1714470801/jiudian-sleep/blob/main/wy1714470801.html

6. 在 Netlify 触发重新部署
https://app.netlify.com/sites/eloquent-mochi-e669ba/deploys
点击 "Trigger deploy" → "Deploy site"
