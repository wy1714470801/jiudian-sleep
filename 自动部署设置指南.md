# 自动部署设置指南（GitHub + Netlify）

## 为什么选择这个方案？

✅ 修改代码后自动部署，无需手动上传  
✅ 有版本历史，可以随时回退  
✅ 完全免费  
✅ 一次设置，终身受益  

---

## 第一步：安装 Git

### Windows 系统：
1. 下载 Git：https://git-scm.com/download/win
2. 双击安装，全部默认选项即可
3. 安装完成后重启终端

### 验证安装：
```bash
git --version
```

---

## 第二步：创建 GitHub 账号

1. 访问 https://github.com
2. 点击 "Sign up" 注册账号
3. 验证邮箱

---

## 第三步：创建 GitHub 仓库

1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 填写信息：
   - Repository name: `fund-query`（或其他名字）
   - Description: `基金净值查询工具`
   - 选择 "Public"（公开）
   - ❌ 不要勾选 "Add a README file"
4. 点击 "Create repository"

---

## 第四步：上传代码到 GitHub

在项目文件夹中打开终端，执行：

```bash
# 1. 初始化 Git 仓库
git init

# 2. 添加所有文件
git add .

# 3. 创建第一次提交
git commit -m "初始化基金查询工具"

# 4. 连接到 GitHub（替换成你的用户名和仓库名）
git remote add origin https://github.com/你的用户名/fund-query.git

# 5. 推送代码
git push -u origin main
```

如果推送时要求登录：
- 用户名：你的 GitHub 用户名
- 密码：需要使用 Personal Access Token（不是账号密码）

### 创建 Personal Access Token：
1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Generate new token → 勾选 "repo" 权限
3. 复制生成的 token（只显示一次，保存好）
4. 在 Git 推送时用这个 token 作为密码

---

## 第五步：连接 Netlify

1. 访问 https://app.netlify.com
2. 使用 GitHub 账号登录（推荐）或注册新账号
3. 点击 "Add new site" → "Import an existing project"
4. 选择 "GitHub"
5. 授权 Netlify 访问你的 GitHub
6. 选择 `fund-query` 仓库
7. 配置构建：
   - Build command: 留空
   - Publish directory: `.`
8. 点击 "Deploy site"

等待 1-2 分钟，部署完成！

---

## 第六步：获取网站链接

部署成功后：
1. Netlify 会生成一个链接，如：`https://random-name-123.netlify.app`
2. 可以自定义域名：Site settings → Domain management → Options → Edit site name
3. 改成：`my-fund-query.netlify.app`

---

## 日常使用：自动部署流程

以后每次修改代码后，只需三条命令：

```bash
# 1. 添加修改的文件
git add .

# 2. 提交修改（描述你做了什么）
git commit -m "优化界面样式"

# 3. 推送到 GitHub
git push
```

推送后，Netlify 会自动检测到更新，自动部署新版本！

整个过程 1-2 分钟完成。

---

## 常用 Git 命令

```bash
# 查看当前状态
git status

# 查看修改内容
git diff

# 查看提交历史
git log --oneline

# 撤销修改（未提交）
git checkout -- 文件名

# 回退到上一个版本
git reset --hard HEAD^
```

---

## 可选：配置 Git 用户信息

首次使用 Git 需要配置：

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

## 故障排除

### 问题1：git 命令不存在
- 安装 Git 后需要重启终端
- 或重启电脑

### 问题2：推送时要求密码
- 不能使用 GitHub 账号密码
- 必须使用 Personal Access Token

### 问题3：推送被拒绝
```bash
# 先拉取远程更新
git pull origin main --rebase

# 再推送
git push
```

### 问题4：Netlify 部署失败
- 检查 GitHub 仓库是否有 index.html 或 wy1714470801.html
- 查看 Netlify 的部署日志

---

## 总结

设置完成后的工作流程：

1. 在本地修改代码
2. `git add . && git commit -m "说明" && git push`
3. 等待 1-2 分钟，Netlify 自动部署
4. 刷新手机上的网页，看到更新

就这么简单！
